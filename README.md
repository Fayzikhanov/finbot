# Family Finance Telegram Bot

Telegram-бот для семейного финансового учета в общем групповом чате.

## Что умеет

- Разбирает расходы и доходы из текстовых сообщений.
- Разбирает голосовые (транскрибация + извлечение операции).
- Ведет статистику по каждому участнику по имени.
- Ведет отдельную статистику семейных расходов (`is_family = true`).
- Хранит данные в SQLite.

## Стек

- Python 3.11+
- `python-telegram-bot`
- OpenAI API
- SQLite

## Установка

1. Создайте и активируйте окружение.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Настройте переменные:

```bash
cp .env.example .env
```

Заполните `.env`:

- `TELEGRAM_BOT_TOKEN`
- `OPENAI_API_KEY`
- опционально модели/валюту/путь к БД

4. Запуск:

```bash
python3 main.py
```

## Настройка Telegram

Чтобы бот видел обычные сообщения в группе, отключите privacy mode в BotFather:

1. `/setprivacy`
2. выберите бота
3. `Disable`

## Онбординг участников

Когда бот добавлен в группу, он представляется как **Семейный финансовый ассистент** и сам спрашивает каждого участника:

```text
@username, как к вам обращаться?
```

Участнику достаточно просто написать имя обычным сообщением.  
`/name` остается как резервный вариант.

## Команды

- `/start` - приветствие и инструкция.
- `/name <имя>` - задать имя участника для статистики.
- `/whoami` - показать текущее имя участника.
- `/members` - список участников и имен.
- `/stats` - общая сводка по участникам за месяц.
- `/mystats` - личная статистика за месяц.
- `/familystats` - общие семейные расходы за месяц.

## Структура

```text
app/
  ai.py        # OpenAI: извлечение из текста и транскрибация голоса
  bot.py       # Telegram handlers и команды
  config.py    # Настройки из env
  db.py        # SQLite схема и запросы
main.py        # Точка входа
```
